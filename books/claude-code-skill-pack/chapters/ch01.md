## この本について

github-flow-kit は Claude Code のスキルパックです。4 つのコマンドをインストールするだけで、毎日の GitHub 作業が劇的に速くなります。

このガイドでは、4 スキルを最大限に活用するための実践的な使い方を解説します。

## 4 スキルの全体像

| スキル | コマンド | 短縮できる作業 |
|---|---|---|
| PR 対応 | `/pr-respond` | レビューコメント返答 (50分 → 3分) |
| リリースノート | `/release-notes` | バージョンごとの変更まとめ (30分 → 2分) |
| Issue 優先付け | `/issue-triage` | バックログの優先度整理 (60分 → 5分) |
| リポジトリ解析 | `/repo-tour` | 初見リポジトリの把握 (30分 → 2分) |

4 スキル合計で、毎週 3〜4 時間のルーティン作業を自動化できます。

## セットアップ (3 分)

前提条件:
- GitHub CLI v2.90.0 以上
- Claude Code (claude-code npm パッケージ)

```bash
# バージョン確認
gh --version    # 2.90.0 以上
claude --version

# 4 スキル一括インストール
for skill in pr-respond release-notes issue-triage repo-tour; do
  gh skill install thinkyou0714/github-flow-kit $skill \
    --agent claude-code --scope user
done
```

インストール完了後、Claude Code を再起動すれば即座に使えます。

## 動作確認

```bash
# インストール済みスキルの確認
gh skill list --agent claude-code

# 出力例:
```
